# say
*Module: [[Misc]]*
Send a message in a specified channel.
#### channel
Channel to send message to
- Type: `CHANNEL`
- Required: `false`
#### edit_msg_id
ID of SlugBot msg to edit
- Type: `STRING`
- Required: `false`
#### do_embed
Should the message be put inside an embed?
- Type: `BOOLEAN`
- Required: `false`
#### schedule_send_time
Delay sending the message; e.g. "4 days 12 hours"
- Type: `STRING`
- Required: `undefined`
