# `/kick`
*Module: [[Moderation]]*<br>
Kick a member from the server
#### member
Member to kick from the server
- Type: `USER`
- Required: `true`
#### reason
Reason for kicking. The reason is DMed to member.
- Type: `STRING`
- Required: `false`
#### context_link
Link to a message or image for extra context
- Type: `STRING`
- Required: `false`
