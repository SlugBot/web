# `/unban`
Unban a member
*Module: [[Moderation]]*
#### user_id
User ID of user to unban
- Type: `STRING`
- Required: `true`
#### reason
Reason for unbanning. The reason is NOT DMed to user.
- Type: `STRING`
- Required: `false`
#### context_url
Link to a message or image for extra context.
- Type: `STRING`
- Required: `false`
