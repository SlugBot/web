# `/mute`
*Module: [[Moderation]]*<br>
Mute a member
#### member
Member to mute
- Type: `USER`
- Required: `true`
#### duration
Duration in english; e.g. "1 hour 20 mins"
- Type: `STRING`
- Required: `false`
#### reason
Reason for muting. The reason is DMed to member.
- Type: `STRING`
- Required: `false`
#### context_url
Link to a message or image for extra context.
- Type: `STRING`
- Required: `false`
