# `/setlevel`
Set the activity level of a member
*Module: [[ActivityExp]]*
## `/setlevel total`
Set the total activity level of a member
#### member
Member to set level for
- Type: `USER`
- Required: `true`
#### level
level to set
- Type: `INTEGER`
- Required: `true`
## `/setlevel weekly`
Set the weekly activity level of a member
#### member
Member to set weekly level for
- Type: `USER`
- Required: `true`
#### level
weekly level to set
- Type: `INTEGER`
- Required: `true`
