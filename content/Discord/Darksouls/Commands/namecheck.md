# `/namecheck`
*Module: [[Darksouls]]*<br>
Check if a name will be censored by the DS3 or ER name filters
## `/namecheck er <name>`
Check against the Elden Ring name filter
#### name
Character name
- Type: `STRING`
- Required: `TRUE`
## `/namecheck ds3 <name>`
Check against the Dark Souls 3 name filter
#### name
Character name
- Type: `STRING`
- Required: `TRUE`
