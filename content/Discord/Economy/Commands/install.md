# `/install`
*Module: [[Economy]]*<br>
Install a process type to increase the total number of that process you can use
## `/install <process> [count]`
#### process
Process type to install
- Type: `Autocomplete Selection`
- Required: `true`
## `/install <process> [count]`
#### count
Number to install
- Type: `INTEGER`
- Required: `false`
