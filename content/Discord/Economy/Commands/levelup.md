# `/levelup`
*Module: [[Economy]]*<br>
Level up a stat
## `/levelup <stat> [number_of_levels]`
#### stat
Stat to level up
- Type: `SELECTION`
  - `firewall`
  - `bypasser`
  - `encryption`
  - `cracker`
  - `scanner`
  - `interceptor`
  - `wire`
  - `injector`
- Required: `TRUE`
#### number_of_levels
If remaining stat levels from a respecc, number of levels to increase stat by
- Type: `INTEGER`
- Required: `FALSE`
