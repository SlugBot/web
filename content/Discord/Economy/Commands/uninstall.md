# `/uninstall`
*Module: [[Economy]]*<br>
Uninstall a process type to free up process slots
## `/uninstall <process> [count]`
#### process
Process type to uninstall
- Type: `Autocomplete Selection`
- Required: `true`
## `/uninstall <process> [count]`
#### count
Number to uninstall
- Type: `INTEGER`
- Required: `false`
