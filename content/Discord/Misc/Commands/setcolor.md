# `/setcolor`
*Module: [[Misc]]*<br>
Customise your name colour
## `/setcolor <color> [member]`
#### color
Hex colour code (e.g. #ff002b) or 'reset' to reset colour
- Type: `STRING`
- Required: `true`
## `/setcolor <color> [member]`
#### member
Set another member's colour
- Type: `USER`
- Required: `false`
