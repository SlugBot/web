# `/userinfo`
*Module: [[Moderation]]*<br>
Get general user information about yourself or another member
## `/userinfo [member] [user_id]`
#### member
Get this member's user info
- Type: `USER`
- Required: `false`
## `/userinfo [member] [user_id]`
#### user_id
Get info on the user with this ID
- Type: `STRING`
- Required: `false`
