# `/unban`
*Module: [[Moderation]]*<br>
Unban a member
## `/unban <user_id> [reason] [context_url]`
#### user_id
User ID of user to unban
- Type: `STRING`
- Required: `true`
## `/unban <user_id> [reason] [context_url]`
#### reason
Reason for unbanning. The reason is NOT DMed to user.
- Type: `STRING`
- Required: `false`
## `/unban <user_id> [reason] [context_url]`
#### context_url
Link to a message or image for extra context.
- Type: `STRING`
- Required: `false`
